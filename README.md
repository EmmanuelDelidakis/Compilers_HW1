Part1:
Το πρόγραμμα Part1 υλοποιεί έναν αναλυτή (parser) και εκτιμητή (evaluator) για μια μικρή γλώσσα 
εκφράσεων που λειτουργεί με χειρισμό συμβολοσειρών και όχι με αριθμητικές τιμές.

Η γλώσσα υποστηρίζει:
-Μεταβλητές (συμβολοσειρές από γράμματα)
-Παρενθέσεις για ομαδοποίηση
-Τελεστές:
    1. / (διαίρεση συμβολοσειρών – αφαίρεση καταλήξης)
    2. ** (δύναμη συμβολοσειρών – διπλασιασμός)

Λειτουργία:
Το πρόγραμμα διαβάζει μία γραμμή από την είσοδο, την αναλύει σύμφωνα με γραμματική χωρίς αριθμούς, και επιστρέφει ένα αποτέλεσμα ως συμβολοσειρά.

Δομή κώδικα:
isLetter(char c)	Ελέγχει αν ένας χαρακτήρας είναι γράμμα (a-z, A-Z)
parseExpression(...)	Ξεκινά την ανάλυση μιας έκφρασης
parseExpressionPrime(...)	Χειρίζεται τον τελεστή /
parseTerm(...)	Αναλύει έναν όρο (term)
parseTermPrime(...)	Χειρίζεται τον τελεστή **
parseFactor(...)	Αναλύει έναν παράγοντα (μεταβλητή ή παρένθεση)
parseAndEvaluate()	Διαβάζει γραμμή, καλεί τον parser, ελέγχει σφάλματα
main(...)	Σημείο εισόδου του προγράμματος

Μεταγλώτιση: javac Part1.java
Εκτέλεση: java Part1
Στην συνέχεια πληκτρολογούμε μια έκφραση όπως a**b**c -> abbcc

Τέλος η γραμματική σε απλή μορφή:
Expression  → Term ExpressionPrime
ExpressionPrime → '/' Term ExpressionPrime | ε
Term        → Factor TermPrime
TermPrime   → '**' Factor TermPrime | ε
Factor      → '(' Expression ')' | Letter+

Part2:
Στο δεύτερο μέρος της εργασίας υλοποιήθηκε ένας parser και translator για μια γλώσσα που υποστηρίζει λειτουργίες πάνω σε strings.

Ο μεταφραστής:
-Διαβάζει πρόγραμμα από stdin
-Αναλύει τη σύνταξη με JavaCUP
-Χρησιμοποιεί JFlex για lexical analysis
-Παράγει ισοδύναμο πρόγραμμα σε Java (stdout)

Η γλώσσα εισόδου υποστηρίζει:
Μόνο String

-Εκφράσεις
Concatenation:
expr + expr

-Συναρτήσεις:
Ορισμός:
f(a, b) {
    expr
}

Κλήση:
f("hello", "world")

-Συνθήκες:
if-else:
if (condition) {
    expr1
} else {
    expr2
}

-Λογικές Συναρτήσεις:
Prefix:
is-prefix-of(a, b)
Suffix:
is-suffix-of(a, b)
